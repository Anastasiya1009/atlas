"Детский Атлас".
Проект будет создан для усвоения информации о истории, географии,
культуре родного края и России в интересной игровой форме.

В данном задании нам важно понимание адаптивности страниц.
Реализуй web приложение, которое показывает все 6 карт при нажатии 
на соответствующий пункт меню. Надписи и объекты на карты наносить 
не нужно. Огромным плюсом будет реализация зуминга. Реализация и 
исполнение зуминга – на твое усмотрение.
Особенности – необходимо создать сайт с адаптивным дизайном, 
который бы корректно отображался на различных мониторах – 16:9, 4:3, 
а так же на мониторах ноутбуков.
